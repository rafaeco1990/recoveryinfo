# Recovery Info SEO Local

Estrutura estática pronta para GitHub Pages.

## O que foi incluído
- Página inicial reestruturada
- 4 páginas de serviço
- 70 páginas locais para cidades do RS
- 3 artigos de blog
- sitemap.xml e robots.txt
- dados estruturados básicos
- CSS e JS leves

## Como publicar no GitHub Pages
1. Suba todos os arquivos para a branch `main`.
2. Em Settings > Pages, selecione `Deploy from a branch`.
3. Escolha `main` e `/root`.
4. Mantenha o arquivo `CNAME` com `www.recoveryinfo.com.br` se for usar domínio próprio.
5. Envie o `sitemap.xml` ao Google Search Console.

## Ajustes pendentes recomendados
